# Hotel Booking System 🏨
Учебный Fullstack-проект формы бронирования отеля.

### 🛠 Технологии:
- **Backend:** PHP (обработка POST-запросов, работа с Cookies)
- **Frontend:** HTML5, CSS3
- **DevOps:** Git/GitHub

### 🚀 Функционал:
- [x] Индивидуальный счётчик посещений на базе Cookies.
- [x] Сбор и обработка данных пользователя (ФИО, адрес, возраст).
- [x] Выбор дополнительных опций (обед, спальное место).
- [x] Валидация согласия на обработку персональных данных.

### 📦 Как запустить:
1. Клонировать репозиторий.
2. Развернуть на локальном сервере (OpenServer/XAMPP).
3. Открыть `booking.php` в браузере.
